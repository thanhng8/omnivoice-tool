👋 Quick start guide — 3 steps

OmniVoice is a zero-shot TTS — there's no fixed catalogue like "F1, F2…". Use it in 3 steps:

1Pick a voice (mode)

Choose one of three: Auto (random voice), Clone (pick from gallery or upload), Design (describe attributes).

New here? Switch to Clone, open 🇻🇳 Vietnamese, click ▶ to preview, then click a voice to select it.
2Type your text

Each line = one audio file. Insert markers like [laughter], [sigh]… from the toolbar.

Each line can override mode/voice/lang individually — useful for dialogue.
3Press Convert All

Audio plays automatically in order. Download per file or click Download all (zip).

Shortcut: Ctrl + Enter

Close this panel once you're familiar with the UI — it'll stay collapsed (saved in localStorage).

🧬 Voice Clone — pick a voice from the gallery below (click ▶ to preview, click the tile to select), or upload your own reference audio (3–10s WAV/MP3). Same profile = same voice every time.
🎨 Voice Design — describe the voice with attributes (gender, age, pitch, accent, dialect). The model is best-trained for English and Chinese; other languages (including Vietnamese) may be unstable — use Clone with the xl_* gallery profiles below instead.

⚙️ Generation parameter details

ParameterDefaultDescription
num_step32Diffusion steps. Higher = better quality + slower. Use 16 for fast preview.
guidance_scale2.0CFG strength. Higher follows instruct/ref more strictly but may distort.
t_shift0.1Time-step shift for the noise schedule.
position_temperature5.0Mask-position randomness. 0 = greedy.
class_temperature0.0Token-sampling randomness. 0 = greedy.
layer_penalty_factor5.0Penalty on deeper codebook layers (encourages low layers first).
speed1.0Speed factor. >1 faster, <1 slower.
durationautoFixed output duration (s). If set, overrides speed.
denoisetruePrepend <|denoise|> token → cleaner audio.
preprocess_prompttrueTrim silence + normalize reference audio.
postprocess_outputtrueTrim trailing silence from output.
